### How to use:
1. Open terminal & clone repo to your pc:
```bash
git clone https://github.com/nikitosikvn1/SEC-Lab-1.git
```
2. Go to your local repo directory:
```bash
cd SEC-Lab-1/
```
3. Run local server:
```bash
go run main.go
```
4. Follow this link http://127.0.0.1:8795/time:
